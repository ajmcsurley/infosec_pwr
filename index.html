Info Security Password Rater
▼

This tool is a password rater written in HTML, CSS, and JavaScript. It makes uses of the zxcvbn library provided by DropBox for password metrics. Type in your test password to the box to view its ratings. Below the metrics you can chart the performance of the password by clicking "Chart it". Enjoy!

CSE 4471 - Information Security; The Ohio State University
Authors - Andrew J. McSurley, Joey Dye, Trevor Stockert, Samuel Maddox, Kyle Gordon
